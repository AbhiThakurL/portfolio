Creating a desktop notifier
Design app in Python

Category:  Technology
Date:  Jan 2024
Author:  Paul Trueman

Creating a desktop notifier app in Python is indeed a fun and useful project! Below, I'll provide a detailed guide to help you build a simple desktop notifier application using Python. This application can notify users about various events, reminders, or updates through pop-up notifications.

"Creativity is intelligence having fun." - Abhishek Thakur
Project Overview

Project Overview

+
-

Use the plyer library to display notifications.

Optionally feature a simple GUI using tkinter.

Allow users to set custom notifications with titles and messages.

Optionally schedule notifications at specific intervals.

Install Required Libraries

You'll need to install a Python library to show system notifications. We will use plyer in this example, which supports cross-platform notifications.

"pip install plyer"
Create a Python Script for the Notifier

You can create a Python file (e.g., notifier.py) with the following code:

# Import required library
from plyer import notification
import time

def show_notification(title, message):
notification.notify(
title=title,
message=message,
app_name="Notifier App",
timeout=10 # Notification will close after 10 seconds
)

# Main function that triggers notifications
def main():
while True:
# Show notification every 60 seconds
show_notification("Reminder", "It's time to take a break!")
time.sleep(60) # Wait for 60 seconds before showing the next notification

if __name__ == "__main__":
main()
Explanation of the Code

Import the necessary libraries:

plyer.notification is used to trigger the desktop notification.
time.sleep() is used to create a delay between notifications.

show_notification() function:
This function triggers the notification with a title, message, and optional parameters like app_name and timeout.

main() function: The main function runs an infinite loop (while True) that triggers the notification every 60 seconds.

if __name__ == "__main__": block: This ensures the script runs when executed directly.

Running the Application

In shop/models.py, define your models. You can start by creating basic models like Product, Category, and Order.

python notifier.py
Customizing the App

Notification Interval: Change the time.sleep(60) to any other value to control the interval between notifications (e.g., time.sleep(3600) for hourly reminders).

Notification Message: Customize the title and message in the show_notification function to show any text you like.

Running in the Background (Optional)

To run the Python script as a background process (especially useful on Linux or Mac), you can add & after the command:

python notifier.py &

On Windows, you can convert this script to a background service using tools like pyinstaller to create an executable, or set up a Task Scheduler task to run it periodically
